Interactive Power BI dashboard built on a Netflix IMDb movies and shows dataset to analyze content performance by release year, runtime, age certification, and engagement. The report includes custom calculated columns for RatingVotes and runtime_hrs, slicers for runtime, release year, and type, and multiple visuals. Used Netflix IMDb dataset with attributes like title, type, release_year, age_certification, runtime, imdb_score and imdb_votes Created calculated column RatingVotes = imdb_score × imdb_votes in Power Query to capture engagement strength. Built DAX column runtime_hrs to convert runtime from minutes to hours for easier interpretation Added slicers for runtime_hrs (Between), release_year (Tiles), and type (Movies/Shows dropdown) to enable interactive filtering Designed donut and pie charts showing yearly counts of movies and shows released after 2010 Created bar chart of movie counts by age_certification since 2000 and combined chart of Rating*Votes and average runtime_hrs by age_certification Arranged visuals into a single‑page report optimized for stakeholders at a media analytics firm to support content acquisition and production decisions
